# Data Structures and Algorithms in C#

This repository contains the exercises and projects I worked on while learning Data Structures and Algorithms in C# with my tutor.

Right now the repository includes:

- Stack implementation using arrays
- Queue implementation using arrays
- Queue implementation using stacks
- Basic testing and stress testing

The goal of this repository is to track my progress while learning DSA concepts and improving my problem solving and C# skills.

---

# Projects

## 1. StackArrayProject

A simple stack implementation built using arrays.

Features:
- Push
- Pop
- Peek
- Dynamic resizing
- Empty checks

Concepts practiced:
- Arrays
- Interfaces
- Dynamic memory resizing
- Stack (LIFO)

## 2. QueueArrayProject

A queue implementation using arrays.

Features:
- Enqueue
- Dequeue
- Peek
- Empty checks

Concepts practiced:
- Queues (FIFO)
- Arrays
- Interfaces
